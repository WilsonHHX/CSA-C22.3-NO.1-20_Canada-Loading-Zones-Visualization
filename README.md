# CSA-C22.3-NO.1-20_Canada-Loading-Zones-Visualization
This interactive KMZ system is built based on CSA C22.3 No.1-20 and integrates all Canadian wind-station data, provincial loading-zone classifications, and the 50-year return-period wind-speed figures. It allows you to check loading requirements directly in Google Earth without switching back and forth between the standard and maps.
