# CSA-C22.3-NO.1-20_Canada-Loading-Zones-Visualization
This interactive KMZ system is built based on CSA C22.3 No.1-20 and integrates all Canadian wind-station data, provincial loading-zone classifications, and the 50-year return-period wind-speed figures. It allows you to check loading requirements directly in Google Earth without switching back and forth between the standard and maps.
The system includes three main components:
* CSA Canada Loading Zones
* Visual layers for Medium A, Medium B, Heavy, and Severe loading regions for each province.
* Climatic Data (Wind Stations)
* All stations from Annex D Table D.2 are included as yellow pins; clicking the points will show the wind-speed values.
50-Year Return Period Wind Speed
* Reference figures from the CSA standard are embedded directly for quick lookup.

Note:
* Wind-station locations and provincial boundaries are approximate and intended for reference only, as the CSA standard does not provide precise geospatial boundaries and wind station location.
